Avaliação Heurística - "Site Feio"

Conceitos Importantes -

As heurísticas de Jakob Nielsen são conceitos que ajudam desenvolvedores de aplicações a resolverem problemas de usabilidade. Há 10 heurísticas no total, mas nem todas são ou precisam se abordadas. Elas são apenas sugestões de design para ajudar na experiência do usuário

Os princípios C.R.A.P. seguem um formato semelhante, utilizados "para produzir um design visualmente mais profissional" (segundo o próprio GitHub de DevLife do Insper). Esses princípios são:

  • Contraste (diferenciação entre objetos na mesma tela);
  • Repetição (elmentos visuais comuns no site);
  • Alinhamento (utilizado para organizar elementos);
  • Proximidade (elementos relacionados entre si devem estar agrupados).
